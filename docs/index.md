features:
  - title: API 请求拦截
    details: 自动拦截页面中的 API 请求并重定向到指定的 Mock URL，支持精确匹配、前缀匹配、正则匹配等多种方式
  - title: 模块化管理
    details: 支持创建多个模块来分类管理不同的 API 配置，支持模块级别的批量开关、重置和导入导出
  - title: Apifox 深度集成
    details: 与 Apifox 深度集成，支持基于 tag 自动同步 Mock 接口，支持迭代管理，自动合并冲突
  - title: 跨插件联调
    details: 支持跨插件批量 QuickMock，外部插件可发送 URL 列表批量生成 Mock 配置，大幅提升联调效率
  - title: 权限管理
    details: 支持一键生成前端权限点，支持按 API 单条复制、按模块批量复制、全量复制
  - title: 双浏览器支持
    details: 同时支持 Chrome 和 Firefox 浏览器，使用同一套代码和配置，多端运行一致体验
