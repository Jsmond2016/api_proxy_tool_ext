# 用户指南

> **更新时间**：2026-07-14；**使用模型**：Codex（GPT-5）；**用户**：Jsmond2016

---

API Proxy Tool 将浏览器页面请求重定向到 Mock 地址，并把接口规则、Apifox 数据、迭代资料和本地存档集中在一个配置页面中。本指南按实际操作目标组织内容。

## 从这里开始

- **第一次使用**：[快速开始](/guide/getting-started)，完成安装并创建第一条 Mock 规则。
- **查找具体功能**：[功能使用指南](/guide/extension-usage)，覆盖 Popup、接口管理、Apifox、存档、权限点和跨扩展联调。
