# 隐私声明

## 数据收集与使用

1. 本扩展不会收集、上传或分享任何用户数据到远程服务器。
2. 所有数据仅在本地浏览器中处理，不会离开用户的设备。
3. 用户配置通过 chrome.storage.sync 保存，仅用于在已登录同一账号的浏览器之间同步配置。
4. 本扩展不会读取或存储用户的浏览历史、书签、密码或其他个人信息。
5. 本扩展不会向任何第三方服务发送数据。
6. 本扩展不执行任何远程代码，所有代码均打包在扩展包内。

## 权限用途

| 权限                            | 用途                                                          |
| ------------------------------- | ------------------------------------------------------------- |
| `activeTab`                     | 获取当前活动标签页的 URL 信息，仅在用户点击扩展图标时临时生效 |
| `storage`                       | 持久化存储用户的模块配置、API 规则和设置项                    |
| `declarativeNetRequest`         | 使用声明式规则拦截和重定向 API 请求，不读取请求/响应体        |
| `declarativeNetRequestFeedback` | 获取规则匹配结果和调试信息，用于展示请求匹配状态              |
| `<all_urls>` (host_permissions) | 支持用户在配置中自定义需要拦截的域名范围                      |

## 数据保留

用户可通过卸载扩展来撤销所有数据访问权限。已保存的配置数据可通过 chrome://extensions 中的扩展选项页面清除。

## 更新

本隐私声明随扩展版本更新。如有重大变更，将在扩展更新说明中告知。
