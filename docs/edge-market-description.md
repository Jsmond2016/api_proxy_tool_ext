# Edge 市场发布描述

## 英文描述

**API Proxy Tool** is a powerful browser extension designed to streamline frontend development by providing seamless API request interception and redirection capabilities. Built with modern web technologies including React, TypeScript, and Vite, this extension offers developers an intuitive solution for managing mock data and API proxies without modifying source code.

### Key Features:

**🎯 Core Functionality**

- **Smart API Interception**: Automatically intercepts API requests and redirects them to specified mock servers
- **Modular Management**: Organize API configurations into multiple modules for different projects or features
- **Flexible Matching**: Supports various URL matching methods including contains, exact match, and regex patterns
- **Response Control**: Configure response delays to simulate real network conditions
- **Direct Mock Responses**: Return mock data directly without requiring external servers

**🔧 Advanced Management**

- **Batch Operations**: Enable/disable entire modules or individual APIs with single clicks
- **Import/Export**: Seamlessly backup and share configurations using JSON format
- **Real-time Search**: Quickly locate APIs by name or URL
- **Global Controls**: Master switch to enable/disable all proxy functionality

**🎨 User Experience**

- **Full-screen Interface**: Comprehensive configuration panel opens in new tabs for optimal workflow
- **Modern UI**: Clean, intuitive interface built with Ant Design components
- **Responsive Design**: Adapts to different screen sizes and resolutions
- **Hot Reload**: Instant updates during development with automatic rebuilds

**🚀 Developer-Friendly**

- **Apifox Integration**: Sync APIs directly from Apifox mock servers using tag-based filtering
- **Permission Management**: Generate and export permission points for backend integration
- **Configuration Templates**: Pre-built examples to get started quickly
- **Cross-browser Support**: Compatible with Chrome, Edge, and other Chromium-based browsers

Perfect for frontend developers. The extension supports Manifest V3 and requires minimal permissions while providing maximum functionality.

**Use Cases:**

- Frontend development with mock data
- API testing and debugging
- Microservices development
- Team collaboration on API interfaces
- Rapid prototyping and demos

Transform your development workflow with API Proxy Tool - the essential extension for modern web development.

## 中文描述

**API Proxy Tool** 是一款专为前端开发设计的强大浏览器扩展，通过提供无缝的API请求拦截和重定向功能来简化开发流程。基于React、TypeScript和Vite等现代Web技术构建，为开发者提供直观的Mock数据管理和API代理解决方案，无需修改源代码。

### 核心功能：

**🎯 主要特性**

- **智能API拦截**：自动拦截API请求并重定向到指定的Mock服务器
- **模块化管理**：将API配置组织到多个模块中，支持不同项目或功能分类
- **灵活匹配**：支持多种URL匹配方式，包括包含匹配、精确匹配和正则表达式
- **响应控制**：配置响应延迟以模拟真实网络环境
- **直接Mock响应**：无需外部服务器即可直接返回Mock数据

**🔧 高级管理**

- **批量操作**：一键启用/禁用整个模块或单个API
- **配置管理**：模块化配置，支持 Apifox 同步（注：JSON 导入导出已移除）
- **实时搜索**：按名称或URL快速定位API
- **全局控制**：主开关控制所有代理功能

**🎨 用户体验**

- **全屏界面**：在新标签页中打开综合配置面板，优化工作流程
- **现代化UI**：基于Ant Design组件构建的简洁直观界面
- **响应式设计**：适配不同屏幕尺寸和分辨率
- **热重载**：开发时自动重建，即时更新

**🚀 开发者友好**

- **Apifox集成**：使用基于标签的过滤直接从Apifox Mock服务器同步API
- **权限管理**：生成和导出权限点用于后端集成
- **配置模板**：预构建示例快速上手
- **跨浏览器支持**：兼容Chrome、Edge和其他基于Chromium的浏览器

非常适合前端开发者使用。该扩展支持Manifest V3，在提供最大功能的同时只需要最少的权限。

**使用场景：**

- 使用Mock数据进行前端开发
- API测试和调试
- 微服务开发
- API接口团队协作
- 快速原型制作和演示

使用API Proxy Tool改变您的开发工作流程——现代Web开发的必备扩展。
