# Chrome Web Store 上架信息

> 以下内容用于 Chrome Web Store 开发者控制台表单填写。

## 摘要（Short Description）

API 请求拦截与 Mock 数据管理工具，支持声明式网络请求重定向。

## 详细描述（Detailed Description）

API Proxy Tool 是一款专为前端开发设计的浏览器扩展，通过声明式网络请求拦截和重定向功能简化开发流程。

**功能特性：**

- 智能 API 拦截：自动拦截 API 请求并重定向到指定的 Mock 服务器
- 模块化管理：将 API 配置组织到多个模块，支持不同项目或功能分类
- 灵活匹配：支持包含匹配、精确匹配和正则表达式多种 URL 匹配方式
- 响应控制：配置响应延迟以模拟真实网络条件
- 直接 Mock 响应：无需外部服务器即可直接返回 Mock 数据
- 批量操作：一键启用/禁用整个模块或单个 API
- 实时搜索：按名称或 URL 快速定位 API
- 配置同步：支持从 Apifox 同步 API 配置

**适用场景：**

- 使用 Mock 数据进行前端开发
- API 测试和调试
- 微服务开发
- API 接口团队协作
- 快速原型制作和演示

## 权限用途

| 权限                            | 用途                                                          |
| ------------------------------- | ------------------------------------------------------------- |
| `activeTab`                     | 获取当前活动标签页的 URL 信息，仅在用户点击扩展图标时临时生效 |
| `storage`                       | 持久化存储用户的模块配置、API 规则和设置项                    |
| `declarativeNetRequest`         | 使用声明式规则拦截和重定向 API 请求，不读取请求/响应体        |
| `declarativeNetRequestFeedback` | 获取规则匹配结果用于界面展示                                  |
| `<all_urls>`                    | 支持用户在配置中自定义需要拦截的域名范围                      |
